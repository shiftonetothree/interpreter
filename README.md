# Interpreter for subset of Pascal
## A implement of [Let’s Build A Simple Interpreter](https://github.com/rspivak/lsbasi) with TypeScript

## 运行最终版本
    npm install
    npm test

## Run latest version
    npm install
    npm test

## 运行所有版本
    npm install
    npm run test-all

## Run all version
    npm install
    npm run test-all
